Matchflare lets you play matchmaker with your friends!

Tech Used:

  • Push notifications via GCM and APNS
  • Persistent image storage via Amazon S3
  • Custom REST API via NodeJS & PostgreSQL
  • Responsive website via Bootstrap + AngularJS
  • Nexmo SMS API + Google Analytics
