Taste

Taste lets you take and share pictures at local businesses in exchange for gifts for you and your friends!
  • Built for both iOS and Android
  • Quickly log-in with Facebook
  • Redeem gifts straight from the Web App
Tech Used:
  • Seamless FB Login & Graph API integration
  • Custom REST API via NodeJS & PostgreSQL
  • Responsive website via Bootstrap + AngularJS
  • Google Analytics & Google Maps API
