Haptic Interface for Improved Surgical Control

Applied Physics Laboratory

Developed intuitive interface between a haptic controller (Sensable Phantom Premium) and a Dexterous Surgical Robot
  • Mapped gestures from 3D Cartesian controller coordinates to Cylindrical robot coordinates
  • Intuitive Force-Feedback
  • Collision Detection
  • Auto-Calibration
Tech Used:
  • Sensable Phantom API
  • Developed in C++ and MATLAB
