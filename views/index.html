Matchflare

go play with matches

Matchflare lets you play matchmaker with your friends!
  • Built for both iOS and Android
  • Match anonymously
  • Register over SMS in seconds
Tech Used:
  • Cross-platform real-time chat server via Websockets
  • Push notifications via GCM and APNS
  • Persistent image storage via Amazon S3
  • Custom REST API via NodeJS & PostgreSQL
  • Responsive website via Boostrap + AngularJS
  • Nexmo SMS API + Google Analytics

Diial

customer service for today

Diial uses the capabilities of smartphones to improve customer support:
  • Replaces IVR ('Press 1 if you are having trouble') with lists of options
  • Replaces elevator hold music with games
  • Autofills personal info (name, address, etc.)
  • Enables in-call photo capture & send (i.e. to troubleshoot your cable layout)
Tech Used:
  • Android SDK
  • Twilio Voice API
  • Real-time 2-way photo & text sending via Websockets
  • REST API > NodeJS & PostgreSQL
  • Interactive Agent Console > AngularJS

Surgical Skill Assessment & Visualization from Tool Motion in Septoplasty

Computational Interaction and Robotics Laboratory

Septoplasty procedures are highly unstructured surgical tasks. This lack of structure makes it difficult to quantify surgical skill. My work:
  • Extracted novel motion-based features for kernel support vector machine to improve automatic classification of surgical skill to 91%
  • Developed real-time 3D tool-tracking visualization tool
  • Built intuitive video segmentation tool to reduce time spent in labeling segments of live-surgical videos
